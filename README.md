# ZLauncher

**ZLauncher** — неофициальный лаунчер Minecraft для Windows.

С ним можно выбирать версии игры, ставить моды, ресурспаки и шейдеры, запускать Minecraft в одном тёмном удобном окне.

> **ZLauncher не связан с Mojang Studios и Microsoft** и не является официальным продуктом.  
> Minecraft — товарный знак Mojang Synergies AB.

## Как установить

1. Скачай **ZLauncher.Setup.exe** (около 50 МБ).
2. Запусти файл. Если Windows покажет предупреждение SmartScreen — нажми **«Подробнее»** → **«Выполнить в любом случае»**.
3. Выбери папку (по умолчанию всё ставится нормально) и нажми **Установить**.
4. Дождись окончания: установщик **сам скачает свежую версию** лаунчера из интернета.
5. Запусти ZLauncher с ярлыка на рабочем столе или из меню Пуск.

**Нужен интернет** при установке — так Setup всегда ставит последнюю версию.

### Если установщик «не запускается»

- Проверь размер файла: должно быть **десятки мегабайт**, а не несколько килобайт.  
  Если файл крошечный — скачай заново (старый файл мог скачаться с ошибкой).
- Скачивай только с [официальной страницы релизов](https://github.com/exteriya1337/ZLauncher/releases/latest).

### Portable

1. Скачай **ZLauncher-Portable.zip**.
2. Распакуй в любую папку.
3. Запусти `ZLauncher.exe`.

## Что умеет лаунчер

- **Версии Minecraft** — релизы, снапшоты и старые версии (по желанию).
- **Сборки** — Vanilla, Fabric, Quilt, Forge, NeoForge, OptiFine.
- **Моды, ресурспаки, шейдеры** — поиск и установка с Modrinth прямо из лаунчера.
- **Свои версии** — можно положить свою папку версии в  
  `%AppData%\ZLauncher\versions\`  
  — она появится в списке.
- **Обычный аккаунт** — ник без входа в Microsoft (офлайн-режим).
- **Автообновление** — см. ниже.

## Обновления

### Установщик

Один раз скачанный **Setup** можно хранить долго.  
При каждой установке он снова тянет **самую новую** версию лаунчера с GitHub.  
Перекачивать Setup после каждого обновления **не обязательно**.

### Сам лаунчер

При **каждом запуске** ZLauncher проверяет, нет ли новой версии на GitHub.

- Если есть новее — обновление **ставится само**, лаунчер перезапустится.  
- Если интернета нет — лаунчер просто откроется как обычно.

## Частые вопросы

**Это официальный лаунчер Mojang?**  
Нет. Это сторонний проект, не связанный с Mojang / Microsoft.

**Нужен ли Java?**  
Нужную Java лаунчер подтянет сам при установке или запуске игры.

**Где лежат файлы игры и настройки?**  
В папке `%AppData%\ZLauncher\` (версии, моды, настройки и т.д.).  
Сам лаунчер по умолчанию: `%LocalAppData%\Programs\ZLauncher\`.

**Можно ли играть с пираткой / офлайн?**  
Лаунчер поддерживает обычный ник (без Microsoft-аккаунта). Онлайн-серверы с обязательной лицензией могут не пускать — это ограничение серверов, не «взлом» Minecraft.

**Безопасно ли скачивать?**  
Скачивай только с этого репозитория GitHub.  
Windows может ругаться на «неизвестный издатель» — это нормально для небольших проектов без платной цифровой подписи.
